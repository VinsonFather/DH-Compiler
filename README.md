# DH+ Compiler

DH+ Compiler 是一个极简主义语言编译器，用于执行一个极简单的任务：当你输入字母 "H" 时，它会输出 "HelloWorld"。这个编译器基于Qt和C++编写，旨在提供一个简洁而直观的用户界面，使代码编译变得轻而易举。

## 功能和特点

- 输入字母 "H" 以获得 "HelloWorld" 输出。
- 用户友好的界面，包括代码输入框和输出框。
- 采用Monospace字体以提供清晰的代码显示。
- 支持代码输入和输出的无换行显示。

## 如何使用

1. 克隆或下载本仓库。
2. 编译并运行程序。
3. 在代码输入框中输入 "H"。
4. 单击 "运行" 按钮，即可在输出框中看到 "HelloWorld"。
